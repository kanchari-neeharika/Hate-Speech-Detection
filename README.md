**Objective**

The main objective of this project is to develop a Hate Speech Detection System using Natural Language Processing (NLP) and Deep Learning. The model analyzes text sentences and classifies them as Hate Speech or Not Hate Speech. This system helps in identifying harmful or offensive language in online communication platforms by automatically analyzing user-generated text.

**Key Insights**
1. **Text Classification**: The model classifies sentences into two categories: Hate Speech and Not Hate Speech.
2. **Text Preprocessing**: Sentences are converted into numerical sequences using a Tokenizer to make them understandable for the neural network.
3. **Padding Sequences**: Text sequences are padded to ensure that all inputs have the same length for efficient model training.
4. **Word Embedding**: An Embedding layer converts words into dense vector representations to capture semantic meaning.
5. **Feature Learning**: The neural network learns patterns in offensive and non-offensive language during training.
6. **Prediction System**: The trained model predicts whether a new sentence contains hate speech along with a confidence score.
